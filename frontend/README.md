# RAG-enterprise Frontend

React + Vite + TypeScript application skeleton.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Quality

```bash
npm run lint
npm run format
npm run test
npm run build
```

## Stack

- React 19
- Vite 8
- TypeScript
- Tailwind CSS v4
- shadcn/ui (New York style)
- ESLint + Prettier
